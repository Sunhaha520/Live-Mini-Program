# Live-Mini-Program
直播小程序，钓鱼直播计分小程序等等，添加绿幕，方便直播，持续更新。
实现功能如下：
- 计时，重置，加减分，胜利场次记录，自定义文字🏆
- 绿色背景，方便直播裁剪👉
- 渐变背景美化，简洁美观✍️
